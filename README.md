# Activity Plan 2026 - Roadmap Podistica

**Live:** [site-xi-dusky-92.vercel.app](https://site-xi-dusky-92.vercel.app)

## Filosofia del Progetto

Questo progetto nasce per pianificare in modo strutturato le attività sportive del 2026. L'obiettivo è costruire una roadmap di gare podistiche selezionate con criterio, bilanciando impegno, distanze e tipologie di percorso.

## Profilo Atleta

- **Disciplina**: Running (strada e trail)
- **Range distanze**: 10 km - 45 km trail
- **Regione di riferimento**: Toscana

## Approccio

1. Raccolta gare disponibili dal [Calendario Podismo Toscana](https://www.calendariopodismo.it/regione/toscana)
2. Selezione delle gare in base a obiettivi e preferenze
3. Creazione calendario personalizzato
4. Pianificazione allenamenti in funzione delle gare scelte
